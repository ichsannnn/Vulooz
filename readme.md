# Vulooz


### Installation

```sh
$ git clone https://github.com/ichsantrueblue/Vulooz vulooz
$ cd vulooz
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan serve
```
